# XOR Encryptor - Двойная версия (GUI + Console)

## 📋 Описание

XOR Encryptor - приложение для шифрования/расшифровки данных с использованием XOR-алгоритма с симметричным ключом.

Доступны две версии:
- **GUI версия** (Avalonia) - полнофункциональный графический интерфейс
- **Console версия** - консольная версия для использования в терминале

## 🚀 Сборка

### Требования
- .NET 8.0 SDK

### Сборка обеих версий

```bash
# GUI версия (выходной файл: XorEncryptor.exe)
dotnet build XorEncryptor.csproj

# Console версия (выходной файл: XorEncryptor.Console.exe)
dotnet build XorEncryptor.Console.csproj
```

### Сборка в режиме Release

```bash
# GUI версия
dotnet build XorEncryptor.csproj -c Release

# Console версия
dotnet build XorEncryptor.Console.csproj -c Release
```

### Публикация standalone приложения

```bash
# GUI версия
dotnet publish XorEncryptor.csproj -c Release -o ./publish/gui

# Console версия
dotnet publish XorEncryptor.Console.csproj -c Release -o ./publish/console
```

## 📱 Использование

### GUI версия

```bash
# Запуск через dotnet
dotnet run --project XorEncryptor.csproj

# Или запуск скомпилированного файла
./bin/Debug/net8.0/XorEncryptor.exe
```

Откроется графический интерфейс с двумя режимами:
1. **Текстовое шифрование** - зашифровать/расшифровать текст
2. **Шифрование файла** - обработать файл целиком

### Console версия

```bash
# Запуск через dotnet
dotnet run --project XorEncryptor.Console.csproj

# Или запуск скомпилированного файла
./bin/Debug/net8.0/XorEncryptor.Console.exe
```

После запуска консольная версия предоставит интерактивное меню:

```
╔════════════════════════════════════════╗
║     XOR Encryptor - Консольная версия   ║
╚════════════════════════════════════════╝

Выберите режим:
  1. Текстовое шифрование
  2. Шифрование файла
  3. Выход

Выбор (1-3): 
```

## 📚 Функции

### Текстовое шифрование

- **Зашифровать**: преобразует текст в Base64-кодированное зашифрованное значение
- **Расшифровать**: восстанавливает исходный текст из Base64-кодированного значения
- Поддержка UTF-8 текста
- В консоли есть опция скопировать результат в буфер обмена (Windows)

### Шифрование файла

- Шифрование файлов любого размера
- Обработка в 80 KB блоках (эффективно для больших файлов)
- Отображение прогресса обработки
- Поддержка любых типов файлов

## 🔐 Безопасность

⚠️ **Важно**: XOR-шифрование - это простейший метод шифрования и **не рекомендуется для защиты критичных данных**. Используйте для:
- Обучения криптографии
- Простого скрытия данных
- Низкоуровневой защиты

Для реальной безопасности используйте современные алгоритмы (AES, ChaCha20 и т.д.)

## 🔄 Особенности

- XOR - **симметричный** алгоритм: encrypt(x, key) = decrypt(encrypt(x, key), key)
- Ключ повторяется циклически для данных больше его длины
- UTF-8 кодировка для текста
- Текстовые результаты кодируются в Base64 для читаемости
- Файлы обрабатываются бинарно без изменения формата

## 💡 Советы

1. **Длинный ключ** - используйте длинные ключи для лучшей безопасности
2. **Сохраняйте ключи** - без ключа расшифровать данные невозможно
3. **Тестируйте файлы** - убедитесь, что расшифрованный файл работает правильно
