# 🎉 XOR Encryptor - Готово к использованию!

## Что было создано

### ✅ Две полноценные версии приложения:

#### 1. **GUI версия** (Graphical User Interface)
- **Файл**: `bin/Debug/net8.0/XorEncryptor.exe` (149 KB)
- **Основа**: Avalonia UI Framework
- **Функции**:
  - Красивый графический интерфейс
  - Режим текстового шифрования
  - Режим шифрования файлов
  - Визуальный прогресс-бар
  - Кнопки для быстрых операций

#### 2. **Console версия** (Консольная)
- **Файл**: `bin/Debug/net8.0/XorEncryptor.Console.exe` (148 KB)
- **Основа**: Консольное приложение (без зависимостей Avalonia)
- **Функции**:
  - Интерактивное меню в консоли
  - Режим текстового шифрования
  - Режим шифрования файлов
  - Красивый вывод с Unicode символами
  - Копирование результата в буфер обмена (Windows)
  - Отображение прогресса обработки файлов

## 🔧 Команды сборки

### Debug версия (по умолчанию)
```bash
# Обе версии
dotnet build XorEncryptor.csproj -c Debug
dotnet build XorEncryptor.Console.csproj -c Debug

# Или используйте скрипт
build.bat debug
```

### Release версия (оптимизированная)
```bash
# Обе версии
dotnet build XorEncryptor.csproj -c Release
dotnet build XorEncryptor.Console.csproj -c Release

# Или используйте скрипт
build.bat release
```

### Standalone публикация
```bash
# GUI версия
dotnet publish XorEncryptor.csproj -c Release -o ./publish/gui

# Console версия
dotnet publish XorEncryptor.Console.csproj -c Release -o ./publish/console
```

## 💡 Особенности Console версии

- **Чистая консоль**: Не требует Avalonia зависимостей
- **Меньший размер**: ~148 KB (против 149 KB GUI, но без лишних зависимостей при standalone)
- **Портативность**: Можно запустить в любой консоли Windows/Linux
- **Интерактивное меню**: Удобная навигация через цифровые выбор
- **Unicode поддержка**: Красивые границы и символы прогресса
- **Быстрая работа**: Почти не требует инициализации Avalonia

## 🔐 Важные замечания

⚠️ **XOR шифрование**:
- Это простейший алгоритм шифрования
- Подходит для обучения и простого скрытия данных
- **НЕ рекомендуется** для защиты критичной информации
- Для реальной безопасности используйте AES, ChaCha20 и т.д.

✅ **Данный проект**:
- Хорошо работает для шифрования документов "от посторонних глаз"
- Быстрая обработка больших файлов
- Симметричное шифрование: ключ для расшифровки = ключ для шифровки
