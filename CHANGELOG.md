# Changelog

Все значительные изменения проекта XOR Encryptor будут документированы в этом файле.

## [1.0.0] - 2026-05-17

### Added
- Инициализация Git репозитория
- GUI приложение на базе Avalonia для шифрования/расшифровки XOR
- Консольная версия приложения для использования в терминале
- XorService для работы с XOR шифрованием
- MVVM архитектура с Community Toolkit
- Поддержка .NET 8.0
- Automated build scripts (build.bat, run.bat)

### Features
- Шифрование и расшифровка данных с симметричным ключом
- Двойная версия: GUI и Console
- Интуитивный интерфейс на Avalonia
- Быстрая обработка больших объемов данных

### Technical Stack
- .NET 8.0
- Avalonia 12.0.1 (GUI)
- Community Toolkit MVVM 8.4.1
- C# 12.0
